\begin{abstract}
	Работа нацелена на создание цифровых двойников с использованием технологии Gaussian Splatting. В качестве целевого объекта выбраны синтетически отрендеренные сцены, движком рендера которых выступает максимально приближенная симуляция (на данный момент) поведения света в плотных средах (Volumetric Monte-Carlo path tracing \refneed).
	
	Проблематика заключается в скорости работы такого алгоритма: работа в реальном времени (вращение, приближение, перемещение по сцене) возможна только на очень мощных видеокартах, которые доступны не всем. Но даже с подходящим оборудованием возникает проблема шума, вызванная вероятностной природой алгоритма. Предлагаемое в работе решение позволяет избавиться от этих недостатков с помощью предварительного долгого "запекания" \refneed отрендеренных сцен для максимальной производительности в дальнейшей работе.
	
	Для целостности картины рассматривается история развития алгоритмов объемного рендеринга (Volume Rendering \refneed), затем один из подходов к решению задачи синтеза новых видов (Novel View Synthesis \refneed) и сам Gaussian Splatting, в рамках той же задачи. В конце теоретической части будет предъявлена мотивация использования GS в поставленной задачи.
	
	Результатами работы является синтетический датасет реалистичных рендеров данных из Visible Human Project \refneed (сегментированная версия \refneed), облака точек как начальная инициализация обучения GS и сама модель Gaussian Splatting.
\end{abstract}
