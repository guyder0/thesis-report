\begin{abstract}
	В современной компьютерной графике одним из ключевых направлений является реалистичный рендеринг, основанный на физически корректной симуляции распространения света. Наиболее распространенным методом в данной области выступает алгоритм трассировки путей Монте-Карло (Monte-Carlo Path Tracing). Однако его применение ограничено высокой вычислительной трудоемкостью и стохастической природой, приводящей к появлению визуального шума на начальных этапах рендеринга. Данные факторы препятствуют визуализации сложных сцен в режиме реального времени.
	
	В качестве решения проблемы для статических и динамических сцен традиционно используются методы предварительного расчета освещения («запекания») в текстуры или воксельные сетки, некоторые из методов описаны, например, в \cite{valve_shading}. В настоящей же работе исследуется применение метода 3D Gaussian Splatting для решения задачи статического «запекания».
	
	В работе проведен ретроспективный анализ алгоритмов объемного рендеринга, рассмотрены современные подходы к синтезу новых ракурсов (Novel View Synthesis) и подробно изучен метод Gaussian Splatting.
	
	Практическая значимость работы заключается в реализации интерактивной модели костной системы человека на основе размеченного набора данных Visible Human Project \cite{segmented_vhp}. Разработанное решение ориентировано на использование в образовательном процессе медицинских факультетов.
	
	В результате работы получены следующие результаты:
	\begin{itemize}
		\item Набор из $700$ синтетических изображений скелета человека, полученных методом трассировки путей с использованием движка Cycles (Blender);
		\item Набор параметров камер и разреженное облако точек, подготовленные для обработки в COLMAP;
		\item Сегментированный (с применением алгоритма KNN) набор облаков гауссиан в формате \texttt{.ply};
		\item Веб-интерфейс для интерактивной визуализации модели с функцией селекции отдельных анатомических структур (костей).
	\end{itemize}
\end{abstract}
